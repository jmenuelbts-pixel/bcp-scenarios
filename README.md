# Scénarios MCV B

Application pédagogique PWA (React + TypeScript + Vite + Supabase) pour Bac Pro
MCV option B.

## Pour reprendre le travail
Lire **`CONTENU_MISSIONS.md`** : tout y est. Le travail restant est de rédiger le
contenu des 56 missions encore vides dans `src/data/contenus.ts` (la mission
`renault-m1` sert de modèle complet).

## Build
1. Créer un fichier `.env` (voir CONTENU_MISSIONS.md pour les valeurs).
2. `npm install`
3. `npm run build` — le dossier `dist/` se déploie sur Netlify par glisser-déposer.
